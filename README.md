# sex-telegram-bot

基于 Telegram 的成人内容分享与互动机器人，支持视频上传、积分系统和分类浏览功能。

## 功能特点

- **视频分享**：用户可上传并分享个人制作的成人视频内容
- **积分系统**：通过分享内容/邀请用户获取积分，解锁高级功能
- **内容分类**：按多样化兴趣需求浏览筛选不同类别内容
- **用户互动**：社区交流功能支持用户分享观感和体验
